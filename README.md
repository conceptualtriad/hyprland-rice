# Hyprland Rice

## Dependencies

- Utilities

  - hyprland
  - kitty
  - spacefm
  - rofi-wayland
  - hyprpaper
  - waybar
  - pywal

- Web browser

  - firefox

- Nerd Font

  - for example, **MesloLGS NF**

## Installation/Usage

1. Install dependencies
1. Copy **.config** and **.themes** into **$Home** directory
1. Launch Hyprland
